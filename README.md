# Icon Creation Guide

## 🎨 Required Icon Sizes

You need to create 4 icon files in different sizes:

- **icon16.png** - 16x16 pixels
- **icon32.png** - 32x32 pixels  
- **icon48.png** - 48x48 pixels
- **icon128.png** - 128x128 pixels

## 🛠️ How to Create Icons

### Option 1: Online Icon Generators
- **Favicon.io** - Upload an image and generate all sizes
- **RealFaviconGenerator** - Professional favicon generator
- **Iconifier** - Simple online icon creator

### Option 2: Design Software
- **Figma** - Free online design tool
- **Canva** - User-friendly design platform
- **GIMP** - Free Photoshop alternative
- **Photoshop** - Professional design software

### Option 3: Use Existing Icons
- **Flaticon** - Free icon library
- **Icons8** - Professional icon sets
- **Feather Icons** - Simple, clean icons

## 🎯 Icon Design Tips

### Recommended Style
- **Simple and recognizable** - Should work at 16x16 size
- **Gradient background** - Use blue/purple gradient to match the extension
- **Break-related symbols** - Coffee cup, timer, pause button, or break symbol
- **High contrast** - Should be visible on light and dark backgrounds

### Color Scheme
- **Primary**: Blue gradient (#667eea to #764ba2)
- **Secondary**: White or light colors for symbols
- **Accent**: Green for active state, orange for paused

### Symbol Ideas
- ⏰ Timer/Clock
- ☕ Coffee cup
- 🎯 Target/Break
- ⏸️ Pause button
- 🧘 Meditation/Relaxation
- 👁️ Eye care symbol

## 📁 File Requirements

- **Format**: PNG (recommended) or JPG
- **Transparency**: PNG with transparent background preferred
- **Quality**: High resolution for larger sizes
- **Naming**: Exact names as shown above

## 🚀 Quick Start

1. **Choose your method** from the options above
2. **Create a simple icon** with your chosen symbol
3. **Export in all required sizes**
4. **Place files in this icons folder**
5. **Reload your extension** in Chrome
