# Quick Installation Guide

## 🚀 Install in 5 Steps

### Step 1: Download
Download or clone this repository to your computer.

### Step 2: Open Chrome Extensions
- Open Google Chrome
- Type `chrome://extensions/` in the address bar
- Press Enter

### Step 3: Enable Developer Mode
- Toggle the "Developer mode" switch in the top right corner

### Step 4: Load Extension
- Click "Load unpacked"
- Select the `take-a-break-extension` folder
- Click "Select Folder"

### Step 5: Start Using
- Pin the extension to your toolbar
- Click the extension icon to configure and start

## 🎯 First Time Setup

1. **Click the extension icon** in your toolbar
2. **Set your preferences**:
   - Break interval: How often you want reminders (20 min recommended)
   - Break duration: How long each break should be (5 min recommended)
   - Enable notifications and sounds
3. **Click "Start Timer"**

## ✅ You're Ready!

The extension will now remind you to take breaks at your chosen intervals. You'll see:
- Desktop notifications
- Beautiful break overlays on web pages
- Break suggestions and timers
- Sound alerts (if enabled)

## 🆘 Need Help?

- Check the main README.md for detailed information
- Ensure all files are present in the extension folder
- Try refreshing the extension if something isn't working
- Check Chrome's extension error page for any issues
